# 518. 零钱兑换II

给定不同面额的硬币和一个总金额。假设每一种面额的硬币有**无限**个。 

写出函数来计算可以**凑成总金额的硬币组合数**。

~~~
示例 1:

输入: amount = 5, coins = [1, 2, 5]
输出: 4
解释: 有四种方式可以凑成总金额:
5=5
5=2+2+1
5=2+1+1+1
5=1+1+1+1+1
~~~

## 完全背包

首先区别：

- 与顺序无关的组合数

- 第 39 题问的是所有的组合列表，应该使用 **回溯算法** 求解；
- 第 518 题问的是组合有多少种，而不是问具体的解。应该使用 **动态规划** 求解。
